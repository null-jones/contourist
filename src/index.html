Contourist

Topographic Joy Plot Generator

How to use
  1. Pick a preset (auto-fetches) or draw a box on the map with Shift + drag, then click Fetch Elevation Data
  2. Adjust the zoom level for more or less terrain detail
  3. Tweak layout, stroke, and style controls
  4. Export as SVG, PNG, or JPG

About the data
Elevation data comes from AWS Terrain Tiles a free, open, global elevation dataset.
